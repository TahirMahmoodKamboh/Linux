**Distribution:**

A distribution or distro is a packaged version of Linux.
Linux is open source, anyone can make a distribution and modify it for a given purpose.
We can interact with Linux OS using Linux GUI and CLI.

****Linux Components****

**Kernel:**
Provides the core functionality of the operating system

**Daemons:**
Provide extra support for the system or the user.

**Examples:**
– syslogd
– sshd

**Applications:**
programs such as text editors, web browsers, and databases.

**Data files:**
Include user content such as text files, music files, and image files.
**Configuration files:**
Contain settings for Linux, services, and applications that are stored in text files.
